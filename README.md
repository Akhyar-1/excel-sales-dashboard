# Excel Sales Dashboard

## Project Overview
This project is an interactive Excel dashboard built to analyze sales performance using Pivot Tables, Pivot Charts, Slicers, and KPIs.

## Features
- Interactive Dashboard
- Sales Analysis
- Monthly Performance
- Regional Performance
- Product-wise Analysis
- Dynamic Filters (Slicers)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
